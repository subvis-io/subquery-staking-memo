# Kusama-Staking-Memo

This subquery will allow you to query the history of staking amount per Era in Kusama.

# Get Started

### 1. install dependencies

```shell
yarn
```

### 2. generate types

```shell
yarn codegen
```

### 3. build

```shell
yarn build
```

### 4. run locally

```shell
yarn start:docker
```
